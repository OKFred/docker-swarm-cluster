# Demo for Docker Swarm Cluster

## Prerequisites

-   [Node.js latest](https://nodejs.org/download/release/latest/)
-   [Docker](https://www.docker.com/)

## Run Server

1. Run `cd ./server`
2. Run `npm run start` or `npm run start:ts` for node v23.6.0

## Run Case

1. Run `cd ./case`
2. Run `npm run start` or `npm run start:ts` for node v23.6.0
